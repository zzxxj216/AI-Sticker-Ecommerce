# 智能推荐系统使用示例

## 场景：批量生成宠物猫贴纸

假设你需要经常为宠物猫生成贴纸，每次都要选择相似的维度（如：玩耍姿态、呆萌表情、户外场景等）。使用智能推荐系统后，可以大幅减少重复操作。

---

## 第一次使用（建立偏好）

### 1. 启动应用
```bash
cd trend_fetcher
python app_gradio.py
```

### 2. 上传图片并分析
- 上传一张猫咪图片
- 输入文本："猫咪，真实"
- ✅ 勾选「🤖 启用智能推荐模式」
- 点击「🔍 分析图片」

### 3. 手动选择维度
系统分析后生成 7 个维度，你手动选择：

```
✓ 猫咪姿态: 玩耍互动 + 好奇探索
✓ 表情神态: 呆萌可爱 + 警觉专注
✓ 场景环境: 户外自然 + 简约纯色
✓ 光线氛围: 暖色调光 + 柔和侧光
✓ 拍摄角度: 俯视角度 + 特写镜头
✓ 猫咪品种: 橘猫/狸花猫 + 布偶/波斯
✓ 整体色调: 暖色调
```

### 4. 生成图片
- 点击「✅ 确认选择」
- 点击「🎯 生成 Prompt」
- 点击「🚀 生成全部图片」

**系统在后台记录了你的选择到「宠物-猫」类别**

---

## 第二次使用（智能推荐）

### 1. 再次上传猫咪图片
- 上传另一张猫咪图片
- 输入文本："猫咪"
- ✅ 确保勾选「🤖 启用智能推荐模式」
- 点击「🔍 分析图片」

### 2. 系统自动推荐
分析完成后，你会看到：

```
### ✅ 分析完成
...

### 🤖 智能推荐（置信度: 100%）
基于你的历史偏好，系统为你推荐了「宠物-猫 常用配置」配置。
你可以直接点击「应用智能推荐」快速应用，或手动选择维度。
```

**所有维度已经自动预选了你上次的选择！**

### 3. 快速应用
两种方式：

**方式 A：直接应用（最快）**
```
1. 点击「✨ 应用智能推荐」
2. 点击「✅ 确认选择」
3. 点击「🎯 生成 Prompt」
4. 点击「🚀 生成全部图片」
```

**方式 B：微调后应用**
```
1. 查看预选的维度
2. 修改 1-2 个不满意的维度
3. 点击「✅ 确认选择」
4. 继续生成
```

## 高级用法

### 1. 查看偏好统计
```python
from agents import PreferenceManager

pm = PreferenceManager()
stats = pm.get_category_stats("宠物-猫")

print(f"总会话数: {stats['total_sessions']}")
for dim_key, dim_stats in stats["dimensions"].items():
    print(f"{dim_key}: {dim_stats['total_selections']} 次选择")
```

### 2. 清除某个类别的偏好
如果你的偏好发生变化，可以清除旧数据：

```python
from agents import PreferenceManager

pm = PreferenceManager()
pm.clear_category("宠物-猫")
print("已清除宠物-猫的历史偏好")
```

### 3. 禁用智能推荐
如果某次生成不想使用推荐：

- 在 UI 中：取消勾选「🤖 启用智能推荐模式」
- 在代码中：
```python
orch = StickerDesignOrchestrator(enable_smart_mode=False)
```

### 4. 导出偏好数据
```python
import json
from pathlib import Path

# 偏好数据存储在
prefs_file = Path("output/user_preferences.json")

# 读取
with open(prefs_file, "r", encoding="utf-8") as f:
    data = json.load(f)

# 备份
import shutil
shutil.copy(prefs_file, "output/user_preferences_backup.json")
```

## 最佳实践

1. **首次使用认真选择**：第一次的选择会影响后续推荐
2. **定期微调**：如果偏好变化，手动调整几次即可更新
3. **分类清晰**：不同类型的内容（猫/狗/人物）会自动分类
4. **保持一致性**：如果你的风格稳定，推荐会更准确
5. **备份数据**：定期备份 `user_preferences.json`

## 常见问题

**Q: 为什么没有显示智能推荐？**
A: 可能是：
- 首次使用该类别（没有历史数据）
- 置信度不足 50%（历史数据太少或太分散）
- 未勾选「启用智能推荐模式」

**Q: 推荐不准确怎么办？**
A:
- 手动调整几次，系统会学习新的偏好
- 或清除该类别的偏好，重新建立

**Q: 数据存储在哪里？**
A: `trend_fetcher/output/user_preferences.json`

**Q: 会影响其他类别吗？**
A: 不会，每个类别的偏好独立存储

**Q: 可以多人使用吗？**
A: 当前版本是单用户，多人使用会混合偏好。未来会支持多用户。
