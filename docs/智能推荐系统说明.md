# 智能推荐系统说明

## 概述

智能推荐系统通过学习用户的历史选择，在后续生成相同类别内容时自动推荐最常用的维度选项，减少重复操作，提升用户体验。

## 核心功能

### 1. 自动类别识别
系统会根据图片分析结果和用户输入文本，自动识别内容类别：
- **宠物-猫**：猫咪相关内容
- **宠物-狗**：狗狗相关内容
- **宠物-通用**：其他宠物
- **人物**：人像、肖像
- **风景**：自然风光
- **食物**：美食、餐饮
- **通用**：其他类别

### 2. 偏好学习
每次用户完成维度选择后，系统会：
- 记录每个维度的选项选择
- 统计选项的使用频率
- 按类别分类存储
- 保留最近 100 条历史记录

### 3. 智能推荐
当用户再次生成相同类别内容时：
- 系统自动分析历史偏好
- 为每个维度推荐最常用的 2-3 个选项
- 计算推荐置信度（覆盖率）
- 如果置信度 ≥ 50%，显示智能推荐

### 4. 快速应用
用户可以：
- **直接应用**：点击「应用智能推荐」一键应用所有推荐
- **手动调整**：查看推荐后手动修改部分维度
- **完全自定义**：忽略推荐，从头选择

## 使用流程

### 首次使用（无历史数据）
```
1. 上传图片 → 分析
2. 手动选择所有维度
3. 系统记录你的选择
4. 生成图片
```

### 后续使用（有历史数据）
```
1. 上传图片 → 分析
2. 系统显示智能推荐（如：「宠物-猫 常用配置」，置信度 85%）
3. 选择操作：
   a) 点击「应用智能推荐」→ 直接确认 → 生成
   b) 查看推荐 → 微调部分维度 → 确认 → 生成
   c) 忽略推荐 → 手动选择 → 确认 → 生成
```

## API 使用

### 在代码中使用

```python
from agents import StickerDesignOrchestrator

# 启用智能模式（默认开启）
orch = StickerDesignOrchestrator(enable_smart_mode=True)

# 分析图片（会自动识别类别并获取推荐）
result = orch.step1_analyze(image_path="cat.jpg", user_text="猫咪")

# 检查是否有智能推荐
if "smart_preset" in result:
    preset = result["smart_preset"]
    print(f"推荐: {preset['preset_name']}")
    print(f"置信度: {preset['confidence']:.0%}")

    # 应用推荐
    orch.apply_smart_preset(preset)

# 或手动选择维度
orch.step2_approve_dimension("pose", ["playful", "curious"], ["玩耍", "好奇"])

# 生成总结（会自动记录偏好）
summary = orch.step2_get_summary()
```

### 偏好管理

```python
from agents import PreferenceManager

pm = PreferenceManager()

# 获取类别统计
stats = pm.get_category_stats("宠物-猫")
print(f"总会话数: {stats['total_sessions']}")

# 获取所有类别
categories = pm.get_all_categories()

# 清除某个类别的偏好
pm.clear_category("宠物-猫")
```


## 优势

1. **节省时间**：减少重复选择，特别适合批量生成相似内容
2. **保持一致性**：基于历史偏好，生成风格更统一
3. **灵活可控**：可以应用推荐，也可以手动调整
4. **自动学习**：使用越多，推荐越准确

## 注意事项

1. **首次使用**：没有历史数据时不会显示推荐，需要手动选择
2. **置信度阈值**：只有当推荐覆盖 ≥ 50% 的维度时才会显示
3. **数据隔离**：不同类别的偏好独立存储，互不影响
4. **隐私保护**：所有数据存储在本地，不会上传
